// getting started · 開始使用

3 steps to install

01
Run the installer
One command. Auto-detects your AI tool. No configuration needed.
# Auto-detect $ npx ux-method-advisor # For specific tools $ npx ux-method-advisor --tool claude-code $ npx ux-method-advisor --tool cursor $ npx ux-method-advisor --tool windsurf # Force update $ npx ux-method-advisor -f
02
Files installed automatically
6 skill folders are created in your project directory.
.agents/skills/ ├── ux-method-advisor/ ← All 65 methods ├── discover/ ← 17 methods ├── define/ ← 21 methods ├── ideate/ ← 13 methods ├── execute/ ← 7 methods └── validate/ ← 7 methods
03
Ask your AI naturally
Skills trigger automatically. No special syntax needed.
# Natural language "我接了一個 revamp project，該怎麼做？" # Slash command "/define 怎樣做用戶訪談？" # English "How do I run a Design Sprint?"
